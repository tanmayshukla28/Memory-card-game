# 🧠 Memory Card Game

An interactive and engaging **Memory Card Matching Game** built using modern web technologies. This project challenges users to test and improve their memory skills by matching pairs of cards in the least number of moves and time.

---

## 🚀 Features

- 🎮 Interactive card-flipping functionality  
- 🧩 Randomized card placement for each game  
- ⏱️ Timer to track user performance  
- 🔢 Move counter for efficiency tracking  
- 🏆 Win detection with completion message  
- 🔄 Restart / New Game option  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Styling:** CSS Flexbox / Grid  
- **Logic:** Vanilla JavaScript (DOM manipulation, event handling)

## ⚙️ How It Works

1. The game board initializes with shuffled cards.  
2. Each card contains a hidden value.  
3. Player flips two cards at a time:  
   - If the cards match → they remain open  
   - If they do not match → they flip back  
4. The process continues until all pairs are matched.  
5. The game displays performance stats like moves and time upon completion.
